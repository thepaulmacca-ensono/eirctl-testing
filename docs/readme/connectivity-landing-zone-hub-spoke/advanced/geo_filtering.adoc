=== Multi-Region Hub Networks

For multi-region deployments, deploy a hub VNet per region and connect them using global VNet peering or Azure Virtual WAN:

[source,hcl]
----
module "hub_uk" {
  source   = "./src/connectivity-landing-zone-hub-spoke/deploy/terraform"
  location = "uksouth"
}

module "hub_eu" {
  source   = "./src/connectivity-landing-zone-hub-spoke/deploy/terraform"
  location = "westeurope"
}
----
