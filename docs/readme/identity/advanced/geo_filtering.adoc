=== Multi-Region Identity

For disaster recovery, deploy identity infrastructure in a secondary region with AD replication:

[source,hcl]
----
module "identity_primary" {
  source   = "./src/identity-landing-zone"
  location = "uksouth"
}

module "identity_secondary" {
  source   = "./src/identity-landing-zone"
  location = "ukwest"
}
----

NOTE: Configure AD Sites and Services to manage replication between regions.
